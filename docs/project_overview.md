# Project Overview

This repository is part of a team project for comparative open chromatin analysis between human and mouse, focusing on adrenal gland and pancreas tissues.

## Current Implementation Scope
**This repository currently implements only Phase 1, Task 1: Human Adrenal Gland open chromatin data quality evaluation.**

Future phases (e.g., cross-species comparison, GO enrichment, motif analysis) are not yet implemented.

## Team Tissues
- Adrenal gland
- Pancreas
