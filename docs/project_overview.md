# Project Overview

This repository is part of a team project for comparative open chromatin analysis between human and mouse, focusing on adrenal gland and pancreas tissues.

## Current Implementation Scope
This repository contains materials and scripts for Phase 1 quality control and downstream comparative analyses.

Implemented workflows include:
- Phase 1 QC outputs and notes
- Task 2 cross-species OCR mapping
- Task 3 conserved versus species-specific OCR comparison
- Task 4 GO enrichment
- Task 5 promoter versus enhancer classification
- Task 6 motif analysis support scripts

Some scripts are still being standardized for portability across user environments. See the main README for current run instructions and path conventions.

## Team Tissues
- Adrenal gland
- Pancreas
